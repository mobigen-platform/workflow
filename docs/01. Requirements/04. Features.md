# Features

- 주요 기능 정의

## 향후 개선 사항

- 웹 UI 추가하여 DAG을 시각적으로 편집 가능하게 개선
- DAG 실행 모니터링 API 추가
- UDF 의존성 관리 기능 추가 (requirements.txt 지원)
- RabbitMQ 데이터 전달 방식 지원 (현재는 XCom/local file 사용)

### UDF

- requirements.txt 를 사용한 디펜던시 추가 가능 해야함

- 입력값 형태 조작 할 수 있어야함
