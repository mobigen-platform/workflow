
## API 목록

### DAG 관리 API

| Method   | Endpoint               | 설명        |
|----------|------------------------|-----------|
| `POST`   | `/api/v1/dag`          | DAG 생성    |
| `DELETE` | `/api/v1/dag/{dag_id}` | DAG 삭제    |
| `GET`    | `/api/v1/dag`          | DAG 목록 조회 |

### UDF 관리 API

| Method   | Endpoint               | 설명        |
|----------|------------------------|-----------|
| `POST`   | `/api/v1/udf`          | UDF 생성    |
| `DELETE` | `/api/v1/udf/{udf_id}` | UDF 삭제    |
| `GET`    | `/api/v1/udf`          | UDF 목록 조회 |
